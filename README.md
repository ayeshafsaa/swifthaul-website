# SwiftHaul Dispatch — Website

A fully responsive, multi-page frontend website for a truck dispatching business built during an internship at DevelopersHub Corporation.

---

## 🚛 Project Overview

SwiftHaul is a professional truck dispatching service website for the Pakistani market. The website allows truck drivers and fleet owners to learn about services, view pricing plans, register as drivers, and contact the company.

---

## 📄 Pages Included

| File | Page |
|---|---|
| `index.html` | Home Page |
| `about.html` | About Us |
| `services.html` | Services |
| `pricing.html` | Pricing Plans |
| `register.html` | Driver Registration Form |
| `contact.html` | Contact / Get a Quote |

---
## Features
- Responsive mobile-friendly design
- Lead generation forms (Quote + Driver Registration)
- 4-step driver registration form with validation
- Pricing plans with monthly/annual toggle
- Interactive FAQ sections
- Google Maps integration on Contact page

## 🚀 How to Run the Project

### Option 1 — Open directly in browser (simplest)
1. Download or clone the repository
2. Open any `.html` file directly in Google Chrome
3. The website will work immediately — no setup needed

### Option 2 — Use VS Code Live Server (recommended)
1. Install [VS Code](https://code.visualstudio.com/)
2. Install the **Live Server** extension inside VS Code
3. Open the project folder in VS Code
4. Right click on `index.html`
5. Click **"Open with Live Server"**
6. Website opens at `http://127.0.0.1:5500`
